## 提供功能

    1. 列表刷新,分页
    2. ReyclerViewAdapter基类，添加headerView、footerView、emptyView
    3. Log,Json,SharedPreference工具类
    4. 网络配置封装，请求结果工具
    5. Activity,Fragment,Presenter基类
    6. 图片加载，缩放工具
